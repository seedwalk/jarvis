# Jarvis Specs

Este directorio guarda la vision del proyecto y la memoria de sesiones.

## Objetivo

Jarvis es un asistente personal flexible para capturar ideas y ejecutar tareas de cualquier tipo.

## Estructura

- `vision.md`: descripcion del producto, alcance y roadmap.
- `product-spec.md`: especificacion funcional de Jarvis (Linear-first).
- `propuesta.md`: propuesta operativa Jarvis + Linear via MCP.
- `sessions/`: historico de contextos de conversacion.
- `templates/session.md`: plantilla para crear nuevas sesiones.

## Convencion de sesiones

Usar formato de nombre:

`YYYY-MM-DD_HHMM_titulo-corto.md`

Ejemplo:

`2026-02-24_1040_plan-memoria-jarvis.md`

Esto facilita orden cronologico y busqueda por tema.
